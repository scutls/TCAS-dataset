# Contributing to TCAS Dataset

Thank you for your interest in contributing to the TCAS dataset! This document provides guidelines for contributing to the project.

## How to Contribute

### Reporting Issues

If you find any issues with the dataset or documentation:

1. Check if the issue already exists in the GitHub issue tracker
2. If not, create a new issue with:
   - A clear and descriptive title
   - Detailed description of the problem
   - Steps to reproduce (if applicable)
   - Expected vs. actual behavior
   - Any relevant screenshots or error messages

### Suggesting Enhancements

We welcome suggestions for improving the dataset or its documentation:

1. Open a new issue with the "enhancement" label
2. Clearly describe the enhancement and its benefits
3. Provide examples or references if possible

### Dataset Corrections

If you find errors in annotations or metadata:

1. Document the error with specific video IDs and frame numbers
2. Provide evidence or reasoning for the correction
3. Submit through the issue tracker with detailed information

### Code Contributions

For code examples, tools, or utilities:

1. Fork the repository
2. Create a new branch for your feature
3. Follow existing code style and conventions
4. Add comments and documentation
5. Test your changes thoroughly
6. Submit a pull request with a clear description

## Code of Conduct

### Our Standards

- Be respectful and inclusive
- Accept constructive criticism gracefully
- Focus on what's best for the community
- Show empathy towards other community members

### Unacceptable Behavior

- Harassment or discriminatory language
- Trolling or insulting comments
- Publishing others' private information
- Other unprofessional conduct

## Questions?

If you have questions about contributing, please:

- Open a discussion in the GitHub Discussions tab
- Contact the maintainers via email
- Check existing documentation and issues first

Thank you for helping improve the TCAS dataset!
